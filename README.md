# CryWasm

Ruby -> Crystal -> Wasm -> Ruby

## Development

Install crystal.
Then: 

```
git clone https://github.com/kojix2/crywasm
cd crywasm
./download-wasm-libs.sh
bundle exec ruby examples/fibonacci.rb
# rake install
```
